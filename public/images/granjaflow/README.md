# Assets GranjaFlow

Pasta reservada para as capturas reais do produto.

- `integrator-dashboard.webp`: painel desktop da integradora.
- `technician-tablet.webp`: visão do técnico em tablet.
- `producer-mobile.webp`: registro do produtor no celular.
- `occurrence-media.webp`: ocorrência com foto ou vídeo.

Os mockups atuais são construídos pela interface e permanecem como fallback até a substituição por capturas aprovadas.
